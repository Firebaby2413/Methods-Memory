Array Methods & Object Methods

What is an Array?

An Array is an object type designed for storing data collections.

Key characteristics of JavaScript arrays are:

  • Elements: An array is a list of values, known as elements.
  • Ordered: Array elements are ordered based on their index.
  • Zero indexed: The first element is at index 0, the second at index 1, and so on.
  • Dynamic size: Arrays can grow or shrink as elements are added or removed.
  • Heterogeneous: Arrays can store elements of different data types (numbers, strings, objects and other arrays).
  • Further documentation: MDN Arrays

Creating an Array:

Arrays are created using the const or let keyword a variable name and [];

const numbers = [1, 2, 3, 4, 5];

let number = [6, 7, 8, 9, 10];

What is an Object?

An object is a variable that contains many variables.

Key characteristics of JavaScript Objects are:

  • Real-life objects: Objects represent real-life objects.
  • Variables: Objects are defined like a variable.
  • Key : Value pairs: Objects are created using key : value pairs.
  • Keys: Keys are the assigned properties of an object.
  • Values: Values are the assigned data associated with the property.
  • Further documentation: MDN Objects

Creating an Object:

Objects are created using the const or let keyword a variable name = {};

const car = {name: "Ford", make: "Escort", year: 1989, mileage: 123,456,};

let car = {name: "Plymouth", make: "Duster", year: 1973, mileage: 178,431,}; "
